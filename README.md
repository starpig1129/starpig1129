## 🔬 Core Research & Agentic Systems

### 🤖 [DATAGEN](https://github.com/starpig1129/DATAGEN)
*An AI-driven multi-agent research assistant designed to automate hypothesis generation, data analysis, visualization, and report writing.*
- **Key Architecture**: Implements a modular agent framework (using LangChain, LangGraph, and OpenAI GPT) coordinating specialized agents for code execution, data visualization, and academic writing.
- **Applications**: Streamlines end-to-end scientific research pipelines, reducing complex manual analysis down to automated iterations.

### 🧪 [research-lab-skills](https://github.com/starpig1129/research-lab-skills)
*A suite of Claude Code skill tools covering the entire academic research lifecycle.*
- **Features**:
  - **Lab Skills**: Structured logs management (`research-log`), pptx report slide generator (`report-slides`), and session routing (`research-mode`).
  - **Academic Research**: Multi-agent literature review agent (`deep-research`), automated paper writer (`academic-paper`), peer review simulator (`academic-paper-reviewer`), and end-to-end research orchestrator (`academic-pipeline`).

### 📡 [PheroPath](https://github.com/starpig1129/PheroPath)
*A stigmergy-based communication protocol for decentralized multi-agent coordination.*
- **Mechanism**: Allows independent AI agents to coordinate, share insights, and signal risks by depositing "invisible pheromones" (metadata/signals) directly on files without modifying their primary content.
- **Concept**: Mimics biological stigmergy to reduce centralized coordination bottlenecks in collaborative agent teams.

## 🛠️ Utility & Interactive Systems

### ☁️ [FileNexus (filetree)](https://github.com/starpig1129/filetree)
*An enterprise-grade, high-performance private cloud file management system.*
- **Tech Stack**: FastAPI (Backend) + React / Vite (Frontend).
- **Features**: Drag-and-drop tree navigation, TUS-based chunked resumable uploads for GB-level files, and seamless Cloudflare Tunnel integration for secure router-less edge deployment.

### 📝 [AutoHomework](https://github.com/starpig1129/AutoHomework)
*An automated homework formatter and LLM-based grading pre-processor.*
- **Functionality**: Automatically converts student homework formats (Jupyter Notebooks, Word, PowerPoint) into clean PDFs and Base64 images to maximize rendering accuracy for downstream LLM grading pipelines. Includes batch-evaluation pipelines.

### ⚔️ [verbal_sparring](https://github.com/starpig1129/verbal_sparring)
*A real-time, WebSocket-based multiplayer verbal battle game featuring an LLM referee.*
- **Architecture**:
  - Backend: FastAPI for game state and WebSockets.
  - Frontend: React / Vite.
  - Core Inference: A local vLLM running a multimodal Gemma model acting as a "toxic referee" to compute damage and return witty critiques. The core inference module is compiled with **Cython** for maximum speed.

### 🐷 [ai-discord-bot-PigPig](https://github.com/starpig1129/discord-LLM-bot-PigPig)
*A feature-rich multi-modal Discord assistant.*
- **Capability**: Integrates large language models for voice interaction, image generation (text-to-image), visual question answering (VQA), conversation history memory, music playback, and channel RAG.

### 📊 [recommendSystem](https://github.com/starpig1129/recommendSystem)
